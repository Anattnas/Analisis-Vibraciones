# Análisis de Vibraciones

Proyecto final - Parte I  
Desarrollado con **Python + Flask + Bootstrap**.

### Descripción
Esta aplicación web muestra gráficas del **acelerómetro** y **giroscopio** según el estado seleccionado:
- Reposo  
- Normal  
- Falla  

### Tecnologías utilizadas
- Flask (Python)
- Bootstrap 5
- Chart.js
- HTML5 / CSS3 / JavaScript

### Instrucciones para ejecutar
1. Activar el entorno virtual.
2. Ejecutar `python app.py`.
3. Abrir el navegador en `http://127.0.0.1:5000`.
